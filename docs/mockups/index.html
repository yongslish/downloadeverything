━━ BACKLOG · V2 探索方向 ━━

Arcade Marquee 真像素

用户参考 "NOW SHOWING · FABLE 5" 提出的方向 —— 点阵 LED 显示屏 + 半色调网点 + 发光字体,是"戏院 marquee / street arcade"美学,和当前 GameBoy 屏内像素不同气质。适合作独立处理页 / 完成庆祝页 / 营销落地页,不占 v1 皮肤名额,列为 v2 探索方向。
